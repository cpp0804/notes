# 1. 例子
从左到右匹配，被匹配过的不会再匹配。例如下面的例子只有0和4，不包括2
```
source: abababa
index:  0123456
expression: aba
```

```java
    /*
    source: abaaaba
    index:  0123456
    pattern:ab
     */
    public static void patternTest1() {
        Pattern p = Pattern.compile("ab");
        Matcher m = p.matcher("abaaaba");
        boolean b = false;
        while (b = m.find()) {
            System.out.print(m.start() + " ");
        }
    }
    /*
    0 4
    */
```

# 2. 正则表达式
```
\d 数字
\s 空格
\w 字符(包括字母、数字、下划线)
例如以下例子返回：0 2 4 5 7 8
source: "a 1 56 _Z"
index:   012345678
pattern: \w

[abc] 匹配a b c
[a-f] 匹配a b c d e f 
[a-fA-F] 匹配a到f之间的和A到F之间的
例如以下例子返回：0 1 4 5 6
source: "cafeBABE" 
index: 01234567 
pattern: [a-cA-C]

0[xX][0-9a-fA-F]：第一个数字是0，第二个字母是x或X，第三个字母在0-9或a-f或A-F之间
例如以下例子返回：6 11
source: "12 0x 0x12 0Xf 0xg"
index:   012345678901234567

+ 一个及以上
* 0个及以上
？ 0个或1个
例如下面的例子返回：0 3 6
source: "1 a12 234b"
pattern: \d+
```
